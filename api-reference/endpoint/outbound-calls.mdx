# Outbound Calls API

## Make a Call

Initiate an outbound call to one or more recipients.

```http
POST /outbound-call
```

### Single Call Request

```json
{
  "to_number": [
    {
      "phone_number": "+91 7874527239",
      "name": "Raju",
      "email": "",
      "company": "",
      "notes": ""
    }
  ],
  "from_number": "+918031272965"
}
```

### Batch Call Request

```json
{
  "to_number": [
    {
      "phone_number": "+91 78745 27239",
      "name": "Raju",
      "email": "",
      "company": "",
      "notes": ""
    },
    {
      "phone_number": "+9199097832",
      "name": "Shyam",
      "email": "",
      "company": "",
      "notes": ""
    }
  ],
  "from_number": "+918031272965",
  "name": "Raju999"
}
```

### Request Parameters

| Parameter    | Type   | Required | Description                                   |
|--------------|--------|----------|-----------------------------------------------|
| to_number    | Array  | Yes      | Array containing recipient details            |
| phone_number | String | Yes      | Recipient's phone number with country code    |
| name         | String | No       | Recipient's name                              |
| email        | String | No       | Recipient's email                             |
| company      | String | No       | Recipient's company                           |
| notes        | String | No       | Additional notes about the recipient          |
| from_number  | String | Yes      | The phone number to call from (must be verified) |
| name         | String | No       | Name for batch call (required for batch calls) |

### Success Response (Single Call)

```json
{
  "success": true,
  "successcode": 200,
  "data": {
    "message": "Call initiated successfully and added to the queue.",

```

## Schedule a Call

Schedule an outbound call for a future date and time.

```http
POST /schedule-call
```

### Single Scheduled Call Request

```json
{
  "scheduled_time": "2025-10-30T12:40:00.587Z",
  "call_request": {
    "to_number": [
      {
        "phone_number": "+91 7874527239",
        "name": "Raju",
        "email": "",
        "company": "",
        "notes": ""
      }
    ],
    "from_number": "+918031272965"
  }
}
```

### Batch Scheduled Call Request

```json
{
  "scheduled_time": "2025-10-30T12:40:00.587Z",
  "call_request": {
    "to_number": [
      {
        "phone_number": "+91 78745 27239",
        "name": "Raju",
        "email": "",
        "company": "",
        "notes": ""
      },
      {
        "phone_number": "+919099999263",
        "name": "Shyam",
        "email": "",
        "company": "",
        "notes": ""
      }
    ],
    "from_number": "+918031272965",
    "name": "Raju999"
  }
}
```

### Request Parameters

| Parameter       | Type   | Required | Description                                   |
|-----------------|--------|----------|-----------------------------------------------|
| scheduled_time  | String | Yes      | ISO 8601 formatted date and time for scheduling the call |
| call_request    | Object | Yes      | Call request object (same as /outbound-call)  |

### Success Response (Single Scheduled Call)

```json
{
  "success": true,
  "successcode": 200,
  "data": {
    "message": "Call scheduled successfully for October 30, 2025 at 12:40 PM UTC.",

      "call_to": "+91 7874527239",
      "call_from": "+918031272965",
      "agent_name": "Test Agent",
      "status": "scheduled"
    }
  }
}
```

| email        | String | No       | Recipient's email address                     |

| notes        | String | No       | Additional notes about the recipient          |
| from_number  | String | Yes      | Caller ID number (must be verified)           |

### Response

#### Success Response (200 OK)

```json
{
  "success": true,
  "successcode": 200,
  "data": {
    "message": "Call initiated successfully and added to the queue.",
    "calls_queued": 1,
    "calls_failed": 0,
    "summary": {
      "total_recipients": 1,
      "successful": 1,
      "failed": 0,
      "is_batch_call": false
    }
  }
}
```

#### Error Response (400 Bad Request)

```json
{
  "success": false,
  "successcode": 400,
  "error": {
    "code": "INVALID_PHONE_NUMBER",
    "message": "The phone number provided is invalid. Please ensure the number includes a valid country code (e.g., +1 for US, +91 for India) and is in the correct format."
  }
}
```

### Error Codes

| Code                 | Description                                                                 |
|----------------------|-----------------------------------------------------------------------------|
| INVALID_PHONE_NUMBER | The provided phone number is invalid or missing country code                |
| UNAUTHORIZED         | Missing or invalid API key                                                 |
| INSUFFICIENT_CREDIT  | Not enough credit to make the call                                         |
| INVALID_CALLER_ID    | The from_number is not a verified caller ID                                |
| RATE_LIMIT_EXCEEDED  | Too many requests. Please wait before making additional calls             |

### Example Usage

```bash
curl -X POST https://api.callintel.com/outbound-call \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to_number": [{
      "phone_number": "+91 7874527239",
      "name": "Raju"
    }],
    "from_number": "+918031272965"
  }'
```

### Notes
- The API supports only one recipient per request for single calls
- Phone numbers must include a valid country code
- The `from_number` must be a verified caller ID in your CallIntel account
- API responses are returned in JSON format
- All timestamps are in UTC timezone
