# API Reference

Welcome to the CallIntel API reference. Use this documentation to integrate with our telephony services.

## Authentication

All API requests require authentication using your API key. Include it in the `Authorization` header:

```http
Authorization: Bearer YOUR_API_KEY
```

## Base URL

```
https://api.callintel.io/v1
```

## Rate Limits

- **Free Tier:** 100 requests per minute
- **Paid Plans:** 1,000+ requests per minute

## Endpoints

### Outbound Calls
- [`POST /outbound-call`](/api-reference/endpoint/outbound-calls) - Make an outbound call
- [`POST /schedule-call`](/api-reference/endpoint/outbound-calls) - Schedule a call for later

### Webhooks
- [Webhook Events](/api-reference/webhooks) - Handle call events with webhooks
